# Инференс — inference/chat.py

Интерактивный чат с моделью в терминале. Поддерживает базовую модель и LoRA-адаптер поверх неё.

## Использование

### Только базовая модель

```bash
python inference/chat.py \
  --base_model "Vikhrmodels/Vikhr-Llama-3.2-1B-Instruct"
```

### С дообученным LoRA-адаптером

```bash
python inference/chat.py \
  --base_model "Vikhrmodels/Vikhr-Llama-3.2-1B-Instruct" \
  --lora_model "./fine_tuning/lora_retrit"
```

### С системным промптом

```bash
python inference/chat.py \
  --base_model "Vikhrmodels/Vikhr-Llama-3.2-1B-Instruct" \
  --lora_model "./fine_tuning/lora_retrit" \
  --system_prompt "Ты помощник бренда ретрита «Познай себя». Пиши тепло, честно, без манипуляций." \
  --temperature 0.3
```

## Параметры

| Параметр | По умолчанию | Описание |
|---|---|---|
| `--base_model` | обязательный | имя базовой модели с HuggingFace |
| `--lora_model` | нет | путь к папке с LoRA-адаптером |
| `--system_prompt` | пусто | системный промпт, задаёт роль модели |
| `--max_new_tokens` | `512` | максимальное количество новых токенов в ответе |
| `--temperature` | `0.3` | температура генерации |

**Температура:**
- `0.1–0.3` — точные, предсказуемые ответы (рекомендуется для Vikhr)
- `0.5–0.7` — баланс между точностью и разнообразием
- `0.8–1.0` — более творческие, но менее стабильные ответы

## Команды в чате

| Команда | Действие |
|---|---|
| `quit`, `exit`, `q` | завершить сессию |
| `clear` | сбросить историю разговора |

## Пример диалога

```
Вы: Напиши текст для сторис — приглашение на ретрит

Модель: Если ты давно откладываешь время для себя — это знак.

Ретрит «Познай себя» — три дня, чтобы выдохнуть, услышать себя
и вернуться в свою жизнь другой.

Места ограничены. Ссылка в описании 🌿

Вы: clear

История очищена.

Вы: quit

До свидания!
```

## Примечания

- Устройство (MPS / CUDA / CPU) определяется автоматически
- История диалога хранит последние 20 сообщений
- Если LoRA-адаптер не указан, чат работает с базовой моделью без дообучения
