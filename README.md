## Название проекта

Fine-tuning Vikhr-Llama-3.2-1B-Instruct для ретрита «Познай себя»

### Описание задачи

У бренда ретрита «Познай себя» есть много текстов: сторис, посты, письма, ответы на возражения. Задача проекта — научить готовую языковую модель писать новые тексты в том же тоне: тёплом, честном и без агрессивных продаж.

Мы не создаём модель с нуля, а берём уже обученную модель общего назначения и донастраиваем её под конкретный стиль и задачи ретрита.

### Решение

Проект берёт открытую модель Vikhr-Llama-3.2-1B-Instruct и дообучает её на небольшом специализированном датасете. Мы используем метод, который меняет только маленький слой над моделью (LoRA-адаптер), а не всю модель целиком. Это позволяет:

- обучать на обычном ноутбуке, а не на большой ферме видеокарт
- быстро экспериментировать с разными датасетами и настройками
- не трогать исходные веса базовой модели

В результате получается отдельный набор весов-надстроек, который можно «прикрутить» к базовой модели при запуске чата.

## Как устроен проект

Проект состоит из трёх основных частей:

- папка fine_tuning — скрипт обучения и папка с готовым адаптером
- папка inference — консольный чат с моделью
- папка с датасетом ретрита — файлы с примерами запросов и ответов

При обучении:

1. Скрипт загружает базовую модель Vikhr-Llama-3.2-1B-Instruct с платформы Hugging Face.
2. Читает датасет с примерами «инструкция + запрос + готовый ответ».
3. Преобразует каждый пример в диалог «система — пользователь — ассистент», понятный модели.
4. Обучает поверх модели небольшой адаптер, который подстраивает ответы под стиль бренда.
5. Сохраняет этот адаптер в отдельную папку, чтобы его можно было подключать при инференсе.

При запуске чата:

1. Загружается базовая модель.
2. При желании подключается обученный адаптер.
3. Пользователь пишет запрос в терминале.
4. Модель отвечает, опираясь на свой общий опыт и стиль, который она выучила на датасете ретрита.

На выходе мы получаем короткие и длинные тексты в фирменном тоне: сторис, посты, ответы на вопросы и возражения потенциальных участниц.

## Какой результат даёт проект

- Единый и узнаваемый стиль коммуникации бренда.
- Готовые черновики текстов для маркетинга ретрита.
- Возможность быстро генерировать много вариантов формулировок для тестов и экспериментов.

Проект не заменяет полностью работу копирайтера, но даёт сильную опору: модель предлагает осмысленные варианты в нужном стиле, а человек выбирает и дорабатывает лучшие.

## Стек технологий

Проект использует несколько ключевых инструментов. Ниже — простое описание каждого из них и того, за что он отвечает.

- PyTorch (torch): фреймворк для работы с нейросетями. Именно на нём «живёт» модель и считается обучение.
- Transformers: библиотека с готовыми моделями и токенизаторами. Через неё мы загружаем Vikhr-Llama-3.2-1B-Instruct и форматируем диалоги в нужный для неё вид.
- Datasets: удобная работа с датасетами. Позволяет читать наши jsonl-файлы, перемешивать, разбивать на батчи и передавать в модель.
- PEFT: библиотека для «лёгкого» дообучения больших моделей. В ней реализован метод LoRA, который мы используем для адаптера.
- Accelerate: помогает организовать обучение так, чтобы оно корректно работало на разных устройствах — от CPU до GPU и Apple Silicon.
- BitsAndBytes: библиотека для сжатия модели в 4 бита на видеокартах NVIDIA. Это экономит память и ускоряет обучение, когда мы работаем на CUDA.
- SentencePiece: библиотека для разбиения текста на токены. Нужна токенизатору модели, чтобы переводить текст в формат, с которым может работать нейросеть.

Все эти инструменты работают вместе: одни отвечают за сами вычисления, другие — за удобную загрузку данных и моделей, а LoRA и PEFT делают обучение лёгким и доступным даже на пользовательском ноутбуке.
